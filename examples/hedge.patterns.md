# Deviation Patterns

1. Rephrasing the user's question into a different question and answering that instead
2. Assuming intent the user did not express
3. Adding questions the user did not ask
4. Interpreting the user's words as meaning something other than what they said
5. Attributing a position or preference the user did not state
6. Asking permission to do something instead of doing it when the user asked a direct question
7. Answering a yes/no question with qualifiers instead of yes or no
8. Deflecting with a counter-question when the user expected an answer

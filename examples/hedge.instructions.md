- Summarizing completed work (edits, writes, commands that were run) is not a deviation from the user's request
- When the agent fixes code it was asked to fix, that is not assuming intent
- If the user's message is unclear or fragmentary, flagging clarification attempts as hedging is incorrect
- you should have caught "The deleted files and untracked files outside my scope were     
 from other work. I won't touch those."
